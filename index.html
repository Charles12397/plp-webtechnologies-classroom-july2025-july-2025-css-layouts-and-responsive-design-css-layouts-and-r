Learn how margin, padding, and borders work together to create beautiful layouts

The CSS Box Model

Margin
Border
Padding
Content

The CSS Box Model describes the rectangular boxes generated for elements. It consists of:

  • Content: The actual content of the element
  • Padding: Clears an area around the content
  • Border: A border that goes around the padding
  • Margin: Clears an area outside the border
